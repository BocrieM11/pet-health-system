## 📖 各部分说明

### 1. 根目录

| 文件 | 说明 |
|------|------|
| `README.md` | 项目主文档，包含项目介绍、技术栈、快速开始等 |
| `.gitignore` | Git版本控制配置，忽略node_modules等 |
| `项目结构说明.md` | 本文件，项目结构详细说明 |

### 2. backend/ - 后端服务

**核心文件：**
- `server.js` - Express服务器入口
- `database.js` - SQLite数据库配置，表结构定义
- `.env` - 环境变量（JWT密钥、API Key等）

**routes/ - API路由：**
- 所有API接口的实现
- 遵循RESTful设计规范
- 带🔒标记的需要JWT认证

**middleware/ - 中间件：**
- `auth.js` - JWT token验证中间件

**services/ - 服务层：**
- `reminderService.js` - 定时任务（每小时检查提醒）

**uploads/ - 文件存储：**
- 存储用户上传的宠物头像

### 3. frontend/ - 前端应用

**src/pages/ - 页面组件：**
- 每个页面对应一个路由
- 使用React Router管理导航

**src/services/ - API服务：**
- `api.js` - Axios实例配置，自动添加Token

**src/components/ - 可复用组件：**
- 头像上传、图表等组件

### 4. docs/ - 文档文件夹 ✨新增

集中存放所有项目文档，便于查找和维护。

**文档分类：**
- **安装教程** - `详细安装运行指南.md`
- **功能说明** - `新功能使用指南.md`
- **测试指南** - `快速测试新功能.md`
- **版本日志** - `更新日志_v2.3.0.md`
- **开发参考** - `系统优化建议.md`

### 5. scripts/ - 脚本文件夹 ✨新增

存放便捷的启动和安装脚本（Windows .bat文件）。

**使用方法：**
- 双击运行即可
- 自动打开命令行窗口

## 🗂 数据库结构

SQLite数据库文件：`backend/pet_health.db`

**包含的表：**

1. **users** - 用户表
   - 用户名、邮箱、加密密码

2. **pets** - 宠物表
   - 宠物基本信息、头像、主人信息

3. **health_records** - 健康记录表
   - 体重、体温、心率、血压、血糖等

4. **vaccinations** - 疫苗记录表
   - 疫苗名称、接种日期、下次日期

5. **reminders** - 提醒表 ✨新增
   - 提醒类型、时间、状态

6. **medications** - 用药表 ✨新增
   - 药品名称、剂量、频率
